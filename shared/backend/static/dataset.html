Dataset Information

E-commerce product review dataset for Elasticsearch query demonstrations

Overview

This dataset contains three related indices representing an e-commerce platform: products, customer reviews, and user profiles. The data is designed to demonstrate various Elasticsearch query types and features.

Total documents: ~6,100 (100 products, 5,000 reviews, 1,000 users)

Usage Tips

  • product_reviews is the default index for most query examples
  • Use review_text and review_title for full-text search
  • Use product_id and review_user_id for exact matches
  • Use review_rating for numeric range queries
  • Use review_date for date range queries
  • Use location in product_users for geo queries
